# Student Performance Analyzer

## Overview
The **Student Performance Analyzer** is a web application that helps teachers and administrators track, analyze, and visualize student performance. It simplifies grade management and provides insights to improve academic outcomes.

## Features
- Add and manage student information (name, ID, class, etc.)
- Record and update exam scores and grades
- Analyze performance by subject, class, or individual student
- Generate charts and reports for easy visualization
- Identify top-performing and struggling students
- Export performance data in CSV format

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Java Servlet
- **Database:** MySQL
- **Tools:** Eclipse IDE / NetBeans, Apache Tomcat
